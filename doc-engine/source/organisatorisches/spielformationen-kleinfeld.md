# Kleinfeldformationen im Jugendfussball

## Übersicht der empfohlenen Formationen

Die folgenden Formationen basieren auf den **WFV-Durchführungsbestimmungen** und **DFB-Ausbildungsrichtlinien** für das
Kleinfeld-Trainingsaufbau (5-gegen-5 und 7-gegen-7).

---

## Formation 2-2 (4 Feldspieler, ohne Torwart)

**Einsatz:** Aufwärmübungen, Pass-Vierecke, Ballkontrolle
**Mannschaftsgröße:** 4 Spieler (beide Teams)

### Taktiktafel: Formation 2-2

```text
                TOR(E)-A
        ┌──────────────────────┐
        │                      │
     R  │      r       r       │
     O  │                      │
     T  │   r             r    │
        ├──────────────────────┤
     B  │   b             b    │
     L  │                      │
     A  │      b       b       │
     U  │                      │
        └──────────────────────┘
                TOR(E)-B

  Legende:
  r     = Roter Spieler
  b     = Blauer Spieler
```

**Merkmale:**

- Flexible Rollen (jeder spielt offensiv und defensiv)
- Ideal für Ballzirkulation und schnelle Kombination
- Fördert Technik und Übergänge
- **Kein Torwart:** Alle Spieler spielen als Feldspieler

## Formation 2-2 mit Torwart (5 Spieler)

**Einsatz:** Kleinfeldtraining Basis, Aufwärmgruppen
**Mannschaftsgröße:** 5 Spieler (inkl. **fliegender Torwart**)

### Taktiktafel: Formation 2-2 mit fliegendem Torwart

```text
                 TOR(E)-A
        ┌──────────────────────┐
        │         tw-r         │
     R  │       r      r       │
     O  │                      │
     T  │   r              r   │
        ├──────────────────────┤
     B  │   b              b   │
     L  │                      │
     A  │       b      b       │
     U  │         tw-b         │
        └──────────────────────┘
                  TOR(E)-B

  Legende:
  r     = Roter Spieler
  b     = Blauer Spieler
  tw-r  = Torwart Rot (fliegend)
  tw-b  = Torwart Blau (fliegend)
```

**Merkmale:**

- **Fliegender Torwart:** Agiert als 5. Feldspieler, spielen beide Teams
- 2 Abwehrspieler pro Team kompakt zentral
- 1 Fliegender TW pro Team (dient als Offensiv- oder Defensivunterstützung)
- **Einsatz bei Turnieren:** Beliebte Aufwärmformation für 5-gegen-5 Gruppenturniere
- **Trainingsschwerpunkt:** Transitions und schnelle Umschalts

**⚠️ Wichtiger Hinweis:**

- Fliegender Torwart macht nur Sinn mit einem **großen Tor** (z.B. D-Jugend Tor in der Feldmitte)
- **Auf Mini-Tore gibt es generell keinen Torwart** – die Mini-Tore dienen nur als zusätzliche Tore neben dem D-Jugend Tor
- Wenn nur Mini-Tore verfügbar sind, spielen alle 5 Spieler als Feldspieler (Formation 3-2 ohne TW)

## Formation 2-3 (6 Spieler mit festem Torwart)

**Einsatz:** Standardformation für strukturiertes Kleinfeld-Spiel
**Mannschaftsgröße:** 6 Spieler (inkl. **fester Torwart**)

### Taktiktafel: Formation 2-3 (Fest-TW)

```text
                 TOR(E)-A
        ┌──────────────────────┐
        │        tw-r          │
     R  │      r       r       │
     O  │                      │
     T  │    r     r     r     │
        ├──────────────────────┤
     B  │    b     b     b     │
     L  │                      │
     A  │      b       b       │
     U  │        tw-b          │
        └──────────────────────┘
                 TOR(E)-B

  Legende:
  r     = Roter Spieler
  b     = Blauer Spieler
  tw-r  = Torwart Rot
  tw-b  = Torwart Blau
```

**Merkmale:**

- **Fester Torwart** pro Team (spielt **nicht** als Feldspieler)
- 2 Abwehrspieler zentral, kompakt beieinander
- 3 Feldspieler offensiv verteilt (zentral + außen)
- **Empfohlen:** D-Jugend Trainings-Standardformation
- **Zonenspiel:** Klare Abschluss- und Verteidigungszonen
- **Trainingsschwerpunkt:** Positionsspiel, Zone-Defense, Abschlussgenauigkeit

## Formation 3-2-1 / Tannenbaum (7 Spieler - offizielle Kleinfeld-Turnier)

**Einsatz:** Offizielle 7-gegen-7 Spielform (Turniere, Meisterschaften)
**Mannschaftsgröße:** 7 Spieler (inkl. **fester Torwart**)

### Taktiktafel: Formation 3-2-1 (Tannenbaum)

```text
                 TOR(E)-A
        ┌──────────────────────┐
        │        tw-r          │
     R  │  r       r       r   │
     O  │                      │
     T  │      r       r       │
        │          r           │
        ├──────────────────────┤
        │          b           │
     B  │                      │
     L  │      b       b       │
     A  │                      │
     U  │  b       b       b   │
        │        tw-b          │
        └──────────────────────┘
                 TOR(E)-B

  Legende:
  r     = Roter Spieler
  b     = Blauer Spieler
  tw-r  = Torwart Rot
  tw-b  = Torwart Blau
```

**Merkmale:**

- **Stabiler Aufbau:** 3er-Abwehrkette sichert defensiv ab
- **Mittelfeld-Kontrolle:** 2 zentrale Spieler organisieren das Spiel
- **Offensiver Fokus:** 1 Spitzenspieler für direktes Spiel und Torablüsse
- **Klassische Turnierformation** für D-Jugend Kleinfeld-Turniere
- **WFV-Standard:** Empfohlene Formation für 7-gegen-7 Wettkampfspiele
- **Trainingsschwerpunkt:** Defensiv-Stabilität, Mittelfeld-Pressing, Konter

## Formation 3-3-2 / 3-3-3 (9 Spieler - D-Jugend offizielle Feldgröße)

**Einsatz:** Offizielle D-Jugend-Spielform (9-gegen-9)
**Mannschaftsgröße:** 9 Spieler (inkl. **fester Torwart**)

### Taktiktafel: Formation 3-3-2 (D-Jugend Standard)

```text
                 TOR(E)-A
        ┌──────────────────────┐
        │        tw-r          │
     R  │ r        r        r  │
     O  │                      │
     T  │    r      r      r   │
        │       r       r      │
        ├──────────────────────┤
        │       b       b      │
     B  │                      │
     L  │    b      b      b   │
     A  │                      │
     U  │ b        b        b  │
        │        tw-b          │
        └──────────────────────┘
                 TOR(E)-B

  Legende:
  r     = Roter Spieler
  b     = Blauer Spieler
  tw-r  = Torwart Rot
  tw-b  = Torwart Blau
```

**Merkmale:**

- **WFV/DFB Standard:** Offizielle Spielform für D-Jugend Wettbewerbe
- **Breite Abwehr:** 3er-Kette mit Sicherheit
- **Dominantes Mittelfeld:** 3 zentrale Spieler kontrollieren das Spiel
- **Flexible Offensive:** 2 Stürmer oder optional erweiterbar auf 3
- **Feldgröße:** ca. 55 m × 35 m bis 70 m × 55 m (mit Abweichungen ±5 m)
- **Trainingsschwerpunkt:** Positionsspiel, Ballzirkulation, schnelle Gegenpresse

## Empfohlener Trainingsaufbau (D-Jugend, 10-15 Spieler Gruppe)

```text
Phase 1: Aufwärmen (Formation 2-2 + Pass-Vierecke)
├─ 4 Kleingruppen × 4 Spieler → Pass-Vierecke 20m × 20m
├─ Fokus: Ballkontrolle, schnelle Pässe
└─ Dauer: 15 min

Phase 2: Technik-Kleinfeld (Formation 2-2-1)
├─ 5-gegen-5 mit fliegendem Torwart
├─ Schwerpunkt: Transitions, Positionsspiel
└─ Dauer: 15 min

Phase 3: Spielform-Training (Formation 2-3 oder 3-2-1)
├─ 6-gegen-6 oder 7-gegen-7 je nach Gruppengröße
├─ Wettkampfform, Zonen-Fokus
└─ Dauer: 20 min

Phase 4: Turnierform (Formation 3-3-2 oder 3-2-1)
├─ 9-gegen-9 oder 7-gegen-7 auf Normalfeld
├─ Offizielle Spielweise vorbereiten
└─ Dauer: 20–30 min
```

## Nomenklatur & Abkürzungen

| Abkürzung | Bedeutung |
|-----------|-----------|
| **TW** | Torwart |
| **TW-Ring** | Fliegender Torwart (agiert als Feldspieler) |
| **Abwehr** | Defensive Spieler zentral oder als Kette |
| **MF** | Mittelfeld (zentrale Ordnung, Ballkontrolle) |
| **Stürmer** | Offensive Spieler, Torschusszone |
| **Flügel** | Außen-Offensive Spieler (links/rechts) |
| **Kompakt** | Spieler stehen eng beieinander (defensive Sicherheit) |

## Zonen-Definition (Kleinfeld 28 m × 30 m)

```text
6 m Abschlusszone (Rot) | = Verteidigungszone (Blau)
        ↓
┌──────────────────────┐
│  Rot (Angriff)       │ Abschlusszone: 0–6 m von Tor A
│                      │
├──────────────────────┤
│  Mittelzone          │ Mittelzone: 6–15 m (9 m breit, 2×)
│  (9 m je Seite)      │
├──────────────────────┤ Mittellinie
│  Mittelzone          │
│  (9 m je Seite)      │
├──────────────────────┤
│  Blau (Abwehr)       │ Abschlusszone: 0–6 m von Tor B
│                      │
└──────────────────────┘
```
